# La app esta corriendo desde el puerto 5000, inicio en la url api/products/

# Para agreagar un producto se debe utilizar una petición  POST, Recibe JSON con un objeto {name:,price:,amount:,descriptio},
en la url api/products/create-product

# Para obtener todos los productos se debe utilizar una petición GET, Responde un Json con un arreglo del documento de MongoDB,
en la url api/products/get-products
